Open Source Projects

Open source Flutter packages by Ishaq Hassan: document scanner, alarm background trigger, assets indexer, and more on pub.dev and GitHub.

Ishaq maintains several open-source Flutter packages published on pub.dev under ishaqhassan.dev:

  • document_scanner_flutter: Native iOS/Android document scanning for Flutter.
  • flutter_alarm_background_trigger: Background alarm scheduler for Flutter.
  • assets_indexer: Asset path generator inspired by Android's R.java.
  • nadra_verisys_flutter: Pakistani NADRA Verisys SDK for Flutter.
  • claude-remote-terminal: Remote-control CLI for Claude Code sessions.
  • goal-agent: Career/goal tracker agent.

In addition, 6 pull requests are merged into the Flutter framework itself. All repositories are public at github.com/ishaquehassan.
