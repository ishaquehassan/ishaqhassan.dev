KEY CHANGES
================================================================================

MENUBAR
  Height: 28px → 24px (thin status bar style)
  Menus: All visible → Only first menu + icons
  Font: 12px → 11px (compact)

DOCK
  Position: Centered floating → Fixed bottom (position: fixed)
  Height: Varies → 60px fixed
  Icons: 44x44px → 36x36px
  Styling: Border-radius 8px → 0 (rectangular)
  Safe area: New - auto padding for notched phones

WINDOWS
  Position: Centered (top:60px, left:50%) → Full-screen (top:24px, bottom:60px)
  Size: Resizable → Fixed full-screen overlay
  Border radius: 12px → 0
  Animation: None → Smooth slide-up/down (300ms)

CONTENT
  Contact grid: 2 columns → 1 column
  Tech grid: auto-fill → 3 fixed columns
  Experience: Readable layout
  Typography: Optimized for mobile (13-14px)
  Padding: 20px → 12px

ANIMATIONS (NEW)
  mobileWindowOpen: 300ms cubic-bezier(0.16, 1, 0.3, 1)
    Transform: translateY(100%) → translateY(0)
  mobileWindowClose: 250ms cubic-bezier(0.5, 0, 0.75, 0)
    Transform: translateY(0) → translateY(100%)

REMOVED/HIDDEN
  Desktop widgets: display: none
  Window min/max buttons: display: none
  Dock tooltips: display: none
  Menu dropdowns (except first): display: none
  Resize handles: display: none

HOW TO TEST
================================================================================

ON DESKTOP:
  1. Open site in browser
  2. Press Ctrl+Shift+M (or Cmd+Shift+M on Mac)
  3. Select device (iPhone 12, Pixel 5, etc)
  4. Observe mobile layout

ON REAL DEVICE:
  1. Visit https://ishaqhassan.dev on iPhone/Android
  2. Observe thin menubar, dock at bottom
  3. Tap dock icon to open window
  4. Watch slide-up animation
  5. Tap close to see slide-down

BROWSER DEVTOOLS:
  1. Open DevTools (F12)
  2. Toggle device toolbar (Ctrl+Shift+M)
  3. Verify media query matches (≤768px)
  4. Check CSS rules apply
